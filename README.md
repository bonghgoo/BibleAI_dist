# 📖 BibleAI (Local Version)

**목회자와 선교사님의 성경 연구를 돕는 지능형 엔진**

BibleAI는 복잡한 성경 주석과 원어 사전을 AI 기술로 통합하여 깊이 있는 말씀 연구를 돕는 도구입니다. 본 저장소는 개인이 소장한 주석 자료를 자유롭게 연동해 사용할 수 있도록 설계된 '엔진'입니다.

---

## ✨ 주요 특징
* **강력한 호환성**: theWord, e-Sword, MySword, 베들레헴성경 등 다양한 주석 모듈 지원.
* **AI 통합 연구**: Groq 및 Ollama를 통한 주석 요약 및 신학적 통찰 제공.
* **보안 및 프라이버시**: 개인의 데이터를 외부 서버에 올리지 않고 로컬에서 안전하게 처리.

## ✨ 2026. 02. 21 업데이트
* **강력한 호환성과 빠른 검색 속도**: 텍스트기반 문서의 검색로직을 개선한 서치엔진 함수 업데이트.  
* **성경역본 지원**: Simple Bible Reader 5.6.2 버전을 이용하여 더워드 등의 성경 모듈을
  로고스바이블의 docx 형식으로 변환하여 사용 가능. 단 성경 태그가 일부 책에서 다르므로 확인 필요.

 ## ✨ 2026. 02. 26 업데이트
* **검색 폴더 설정하지 않고** 에이전트 검색 시 무한루프 버그 수정.
* **e-sword cmti, cmtx** 모듈 검색 오류 수정.
* **바구니 담기** 건수 표시 오류 수정.

## 🚀 시작하기
자세한 설치 방법은 파일 목록의 **[설치가이드.txt]**를 확인해 주세요.

1. 이 저장소의 코드를 내려받습니다.
2. `1_필수부품설치.bat`를 실행하여 환경을 구축합니다.
3. `dct` 및 `commentaries` 폴더에 본인의 자료를 넣습니다.
4. `2_프로그램실행.bat`으로 실행합니다.
