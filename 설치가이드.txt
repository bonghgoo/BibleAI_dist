2. 나만의 주석/사전 라이브러리 구축법
-----------------------------------------------------------
본 프로그램은 '빈 그릇'과 같습니다. 위 웹앱 데모에서 보신 다양한 자료들을 
로컬 컴퓨터에서 사용하시려면, 개인이 소장한 파일을 아래 폴더에 넣으시면 됩니다.

① 원어 사전 연동: [dct] 폴더에 넣으세요.
   - 지원 확장자: *.dct.twm (theWord용 사전)

② 성경 주석 연동: [commentaries] 폴더에 넣으세요.
    폴더명은 사용자가 임의로 정해도 자동 인식됩니다. 
   - 지원 확장자: 
     * theWord용 주석: *.cmt.twm
     * e-Sword용 주석: *.cmti, *.cmtx
     * MySword(안드로이드)용 주석: *.cmt.mybible
     * 베들레헴(크로스웨이)용 주석: *.cdb
     
   ※ 주의: 
    1. 주석 모듈만 인식합니다. 성경 본문 모듈이나 일반 도서, 
     그래픽 모듈 등은 프로그램 엔진에서 인식되지 않으니 참고 바랍니다.
    2. 안드로이드나 아이폰용 mybible 앱과 혼동하지 마세요. 해당 앱들의 모듈은
     확장자가 *.sqlite3 입니다. 대신 이 모듈들을 Simple Bibe Reader로 인식되는
     더워드용 모듈 등으로 변환하면 사용 가능합니다.

-----------------------------------------------------------
3. 주요 실행 파일 안내
-----------------------------------------------------------
* 1_필수부품설치.bat : 최초 1회 실행 (파이썬 라이브러리 자동 설치 및 API 설정)
* 2_프로그램실행.bat : 설치 완료 후 매일 실행하는 프로그램 아이콘
* main.py : 프로그램의 심장부 (소스 코드)
* core/ : 데이터 처리를 담당하는 핵심 엔진 폴더
* 설치 폴더에서 터미널을 열고 streamlit run main.py 명령으로도 실행됩니다.
